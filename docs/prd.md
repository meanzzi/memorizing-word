# Product Requirements Document (PRD): 영어 암기장 웹앱

## 1. 프로젝트 개요
사용자가 영어 단어를 효율적으로 저장하고 암기할 수 있도록 돕는 간단한 웹 기반 애플리케이션입니다. 브라우저의 로컬 스토리지를 활용하여 별도의 서버 없이도 개인화된 단어장을 유지할 수 있습니다.

## 2. 주요 목표
- 직관적인 단어 관리 인터페이스 제공
- 다양한 퀴즈 모드를 통한 학습 효과 극대화
- 모바일 환경 최적화 및 접근성 강화 (다크 모드 등)

## 3. 핵심 기능 (Functional Requirements)

### 3.1. 단어 관리 (CRUD)
- **추가**: 단어(영어), 뜻(한글)을 입력하여 저장.
- **조회**: 저장된 단어 목록을 리스트 형태로 확인.
- **수정**: 기존 단어나 뜻을 수정.
- **삭제**: 단일 단어 혹은 선택된 단어를 삭제.

### 3.2. 학습/퀴즈 기능
- **객관식 테스트**: 무작위로 제시된 단어의 뜻을 카드 중에서 선택.
- **주관식 테스트**: 제시된 뜻을 보고 해당 영단어를 직접 타이핑하여 입력.
- **발음 듣기**: Web Speech API 등을 활용하여 영단어 발음 재생.

### 3.3. 설정 및 기타
- **다크 모드**: 사용자 선호에 따른 테마 전환 기능.
- **데이터 저장**: 모든 데이터는 브라우저의 `localStorage`에 JSON 형식으로 저장.

## 4. 사용자 경험 및 인터페이스 (UX/UI)
- **모바일 우선 (Mobile-First)**: 모바일 기기에서 사용하기 편리한 레이아웃 및 버튼 크기 설계.
- **미니멀 디자인**: 암기에 집중할 수 있는 깔끔하고 직관적인 UI.
- **반응형 웹**: 데스크톱 환경에서도 어색하지 않은 레이아웃 제공.
- *상세 디자인은 `docs/design.md`에서 추후 정의.*

## 5. 기술 스택 (Technical Stack)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: Browser LocalStorage
- **APIs**: Web Speech API (발음 기능)

## 6. 비기능적 요구사항 (Non-Functional Requirements)
- **로그인 불필요**: 복잡한 가입 절차 없이 즉시 사용 가능.
- **오프라인 사용 가능**: 서버 통신이 필요 없으므로 인터넷 연결 없이도 기본 기능 작동.
- **성능**: 빠른 페이지 로딩 및 부드러운 전환 효과.

## 7. 향후 확장 가능성 (Future Scope)
- 단어장 카테고리 분류 기능.
- 암기 진척도 시각화 (통계).
- 외부 데이터(CSV 등) 가져오기/내보내기 기능.
