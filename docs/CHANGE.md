# Change Log: 영어 암기장 웹앱

## [1.0.0] - 2026-05-05

### 추가된 기능
- **프로젝트 기초 설정**: HTML5 뼈대 구성, Pretendard 폰트 및 Material Symbols 아이콘 연동.
- **테마 시스템**: 라이트 모드 및 다크 모드 지원 (CSS 변수 활용).
- **데이터 관리 (StorageManager)**: 브라우저 `localStorage`를 이용한 영구 데이터 저장 로직 구현.
- **발음 기능 (SpeechEngine)**: Web Speech API를 활용한 영단어 TTS 기능 추가.
- **사용자 인터페이스 (UIManager)**: 토스 스타일의 깔끔한 UI 구성, 모바일 최적화 레이아웃 및 애니메이션 적용.
- **단어 관리 (CRUD)**: 단어 추가(모달), 목록 조회, 수정, 삭제 기능 구현.
- **퀴즈 시스템 (QuizEngine)**: 
    - 객관식(4지 선다) 및 주관식(빈칸 입력) 퀴즈 무작위 생성.
    - 퀴즈 진행률 표시(프로그레스 바) 및 결과 화면 구현.
- **UI/UX 폴리싱**: 부드러운 화면 전환, FAB 버튼, 반응형 레이아웃 최종 조정.
- **애니메이션 최적화**: 다크 모드 전환 시 헤더와 바디의 전환 속도를 일치시켜 시각적 어색함 해결.
